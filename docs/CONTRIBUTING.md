# Contributing

Public repositories for TKB Strategies contain only generalizable, reusable work: frameworks, templates, plugin source code, and workshop outlines with placeholder content.

Never commit client-specific content. That includes client names, dollar amounts, personal stories, or engagement-specific data.

If content references a specific organization or a person's confidential information, it stays in Google Workspace and out of this repository.

Before committing, ask: "Could I share this publicly without compromising any client or individual?" If the answer is no, it does not belong here.

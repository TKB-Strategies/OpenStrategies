# Stack

## Build And Deploy

- VS Code for local editing and repository management.
- Claude Code and OpenAI Codex for AI-assisted development workflows inside the repository.
- GitHub for version control, collaboration, pull requests, and organization-level visibility.
- Softaculous Git on Namecheap for pulling repository changes into WordPress environments.
- GitHub Actions for automated FTP deployments where Git pull is not the deployment path.

## Create And Publish

- Google Workspace for client work, proposals, and internal documents. Client-specific material never belongs in this repository.
- Canva for visual assets and brand materials.
- Audacity for podcast production, including "Breaking the Binary."
- Readsy for content distribution.

## Sell And Distribute

- Gumroad for digital products on the free tier.
- Udemy for courses on the free tier.
- The Movement App for community distribution on the free tier.

## Hosting

- Namecheap shared hosting with cPanel.
- WordPress with Divi.
- MySQL for application data.
- SSL managed through cPanel.

## Repository Boundary

Google Workspace is the private operational layer. GitHub is the public-facing layer. These two never cross for client-specific content.

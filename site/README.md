# TKB Strategies Documentation Site

This directory contains the [Docusaurus](https://docusaurus.io/) project that powers the public documentation site for TKB Strategies.

**Live site:** [tkb-strategies.github.io](https://tkb-strategies.github.io)

## Quick Start

```bash
npm install
npm run start
```

## What This Site Publishes

- Open-source consulting frameworks (Compassionate Agility, Liberation Mapping, Steward's Manual)
- Workshop and presentation templates
- Plugin and tool documentation
- Project updates and building-in-public narrative

## What Does NOT Go Here

Client-specific data, engagement details, financials, or confidential information. See [CONTRIBUTING.md](../docs/CONTRIBUTING.md) for the full policy.

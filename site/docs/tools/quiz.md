# Patterns & Protection

> Understanding Your Workplace Responses

---

A custom-built self-reflection tool that helps mission-driven professionals identify how their nervous system patterns — fight, flight, freeze, fawn, flop — shape leadership decisions and workplace responses.

## What It Measures

The tool combines two assessment components: an adapted ACE (Adverse Childhood Experiences) score providing context on early life patterns, and a workplace scenario-based assessment identifying dominant protective response patterns in professional settings.

## Technical Details

Built as a standalone WordPress plugin (`tkb-quiz-tracker-v2`) deployed on [tkbstrategies.com](https://tkbstrategies.com/patterns-and-protection). The plugin uses WordPress AJAX API for data capture, `dbDelta()` for database table management, and includes an admin dashboard for response analytics.

- **15 questions** · Instant results · No account required
- Trauma-informed design with pause-and-care-for-yourself guidance
- Anonymous completion with optional email delivery of results
- Workplace Regulation Toolkit included with results

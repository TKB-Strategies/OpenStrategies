# TKB Strategies

TKB Strategies is a human-centered strategy consulting firm specializing in liberation-centered organizational development, executive coaching, and capacity building for mission-driven organizations.

This repository contains open-source frameworks, tools, and WordPress plugins developed as part of an open consulting company model. Client-specific data and engagement details are never stored here.

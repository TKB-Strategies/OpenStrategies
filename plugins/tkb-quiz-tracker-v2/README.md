# tkb-quiz-tracker-v2

This directory contains the source code for the custom WordPress plugin that supports the public quiz experience and data capture workflow.

Belongs here: plugin PHP, JavaScript, CSS, documentation, and deployment-safe assets that are generalizable and intended for WordPress source control.

Does not belong here: quiz responses, exports, database dumps, credentials, `.env` files, or any client-specific data tied to a real organization or individual.

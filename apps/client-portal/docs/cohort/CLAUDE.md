# cohort/ — Coachee Profiles and Sessions

Each coachee gets a subdirectory: cohort/{name-slug}/
Inside: a profile page (index.md) and a sessions/ folder.

Profile pages are generated from the coaching-profile.md template
and updated when the roster sheet changes. Session notes use the
session-note.md template.

## Coaching Spine Phases
1. Stabilize — Reduce emotional charge
2. Name — See patterns, not problems
3. Structure — Clarity and containment
4. Practice — Confidence through use
5. Sustain — Anchor identity

The spine-badge CSS classes render colored phase indicators:
spine-badge--stabilize, spine-badge--name, etc.

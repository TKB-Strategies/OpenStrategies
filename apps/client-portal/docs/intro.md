<div className="welcome-page__eyebrow">Client Engagement Knowledge Base</div>

This site is the shared home for the work as it unfolds. It holds phase
materials, coaching notes, tools, and context in one place so the engagement
can deepen without scattering its memory across folders, decks, and email
threads.

<p className="welcome-page__lede">
  Use the sidebar to move through the engagement spine, revisit session
  materials, and return to decisions with enough context to act on them well.
</p>

## How To Use This Space

Start with the current phase, then move into the cohort or tools sections as
needed. The structure is designed to support regular use over time, not a
single review moment.

| Area | What it holds |
| --- | --- |
| Overview | A shared orientation to the engagement |
| Phases | Working notes, materials, and deliverables by phase |
| Cohort | Profiles and session records for participants |
| Tools & Assessments | Frameworks, prompts, and supporting resources |

| Signal | Current template setup |

# docs/ — Published Content

Everything in this directory becomes a page on the client's Docusaurus site.
Phase folders (01-stabilize/ through 05-sustain/) correspond to the coaching
spine. The cohort/ directory holds per-coachee profiles and session notes.

## Conventions
- Use frontmatter: title, sidebar_label, sidebar_position
- Write in human-centered, liberation-rooted language
- Never include internal coaching notes (those go in _config/internal/)
- Link to tools-assessments/ for any referenced frameworks

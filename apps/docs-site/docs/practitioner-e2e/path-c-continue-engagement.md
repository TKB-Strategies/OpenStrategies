# Path C: Continue An Existing Engagement

Use this path on an active engagement that already has accumulated state. This
is the weekly operation check.

## 1. Open The Engagement

Sign in as the practitioner and open the active engagement.

Expected:

- The engagement overview loads.
- The Attention strip reflects current state.
- The status strip counts agree with the visible work.

## 2. Triage Attention

Work through the Attention strip. Click into each item and resolve or defer it.

Expected:

- Attention items link to the correct sub-route.
- Resolved items disappear or change state after refresh.
- Deferred items remain understandable.

Stop if the strip is empty while an obvious issue exists, such as an unsigned
agreement, open critical risk, or overdue closure survey.

## 3. Walk The Tabs

Review the active engagement surfaces:

- **Work items**: review open items and archive stale ones.
- **Risks**: re-rate current risks.
- **Deliverables**: advance in-flight items toward delivered.
- **People**: review submitted forms.
- **Sessions**: confirm the next session.
- **Source materials**: sync new partner files from engagement detail or
  `/admin/drive`.
- **Timeline**: confirm the work appears as activity events.

Expected:

- Every sub-route loads.
- Actions write activity.
- Timeline reflects what happened during the session.
